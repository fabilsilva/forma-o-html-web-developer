História da Internet

(voltar)

A história da internet é fascinante e cheia de marcos importantes. A ideia inicial surgiu durante a Guerra Fria, na década de 1960, quando o Departamento de Defesa dos Estados Unidos decidiu criar uma rede de comunicação que pudesse resistir a ataques militares. Assim nasceu a ARPANET (Advanced Research Projects Agency Network), em 1969. A primeira comunicação bem-sucedida entre computadores ocorreu em 29 de outubro de 1969, conectando a Universidade da Califórnia, em Los Angeles (UCLA), ao Instituto de Pesquisa de Stanford.

Nos anos 1970, surgiram os protocolos de comunicação TCP/IP, desenvolvidos por Vint Cerf e Bob Kahn. Esses protocolos permitiram que diferentes redes de computadores se comunicassem entre si, formando uma verdadeira "rede de redes". Em 1983, a ARPANET adotou oficialmente o TCP/IP como padrão, marcando um passo significativo na evolução da internet.

A década de 1990 foi um período de grande transformação para a internet. Em 1989, Tim Berners-Lee, um cientista britânico, propôs a criação da World Wide Web (WWW). Sua invenção permitiu a interconexão de documentos através de hyperlinks, tornando a navegação e o acesso à informação muito mais intuitivos. Em 1993, o lançamento do navegador Mosaic popularizou o uso da web, trazendo milhões de novos usuários para a internet.

Ao longo dos anos 2000, a internet continuou a se expandir e evoluir, com a explosão das redes sociais, o aumento da velocidade das conexões e o surgimento de novas tecnologias, como os dispositivos móveis e a computação em nuvem. Hoje, a internet é uma parte essencial da vida cotidiana, transformando a maneira como trabalhamos, nos comunicamos e consumimos informações.

Sobre o HTML

(voltar)

HTML (HyperText Markup Language) é a linguagem fundamental para a criação de páginas web. Criada por Tim Berners-Lee no final dos anos 1980, o HTML permite que desenvolvedores estruturassem e organizassem conteúdo na web de maneira eficiente e acessível. O HTML é uma linguagem de marcação que utiliza "tags" para definir diferentes elementos de uma página, como títulos, parágrafos, imagens, links, tabelas, entre outros.

Cada tag HTML é delimitada por sinais de menor e maior (< e >), e muitas tags possuem uma tag de abertura e uma tag de fechamento. Por exemplo, a tag <p> é usada para definir um parágrafo e é encerrada com </p> . O HTML também permite a incorporação de outras tecnologias, como CSS (Cascading Style Sheets), que é usado para definir o estilo e layout das páginas, e JavaScript, que adiciona interatividade e funcionalidades dinâmicas.

Com o passar dos anos, o HTML passou por diversas revisões e atualizações. A versão mais recente, HTML5, introduziu novos elementos e funcionalidades, como suporte nativo para áudio e vídeo, armazenamento local de dados e uma estrutura semântica melhorada. Esses avanços tornaram o desenvolvimento web mais poderoso e flexível, permitindo a criação de sites e aplicativos web modernos e interativos.

O Que São As tags HTML

são os blocos de construção fundamentais de qualquer página web. Elas funcionam como instruções que informam ao navegador como exibir e organizar o conteúdo. Cada tag tem um propósito específico e pode ser combinada com outras tags para criar estruturas complexas e ricas.

Aqui estão algumas das tags mais comuns e suas funções:

  • <h1> a <h6>: Definem os títulos de diferentes níveis, onde <h1> é o mais importante e <h6> o menos importante. Esses títulos são usados para organizar o conteúdo e melhorar a acessibilidade.
  • <p>: Define um parágrafo de texto. É uma das tags mais básicas e frequentemente usadas no HTML.
  • <a>: Cria um link (âncora) que pode levar a outra página, um recurso ou uma parte específica da mesma página. A tag <a> requer o atributo href para especificar o destino do link.
  • <img>: Insere uma imagem na página. A tag <img> precisa do atributo src para especificar o caminho da imagem e pode incluir outros atributos, como alt para fornecer uma descrição alternativa.
  • <div>: Agrupa conteúdo em blocos, útil para layout e estruturação da página. A tag <div> é frequentemente usada em conjunto com CSS para aplicar estilos a seções específicas da página.
  • <ul>, <tol> e <li>: Criam listas não ordenadas (bullet points), ordenadas (numeradas) e itens de lista, respectivamente. Essas tags são usadas para organizar informações em formato de lista.

Além dessas, há muitas outras tags no HTML, cada uma com suas próprias funcionalidades e usos específicos. O conhecimento e uso adequado das tags é essencial para criar páginas web estruturadas, acessíveis e visualmente agradáveis.

Outro exemplo de uma tag diferente é a <del>, que funciona para mostra textos deletados.
